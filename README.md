# README

This is a highly provisional AI translation of the python library `refspy`.

The [NOTEBOOK_JS.ipynb](NOTEBOOK_JS.ipynb) file illustrates usage; this is a direct translation of the [NOTEBOOK.ipynb](https://github.com/eukras/refspy/blob/master/NOTEBOOK.ipynb) file from Refspy.

The Jupyter notebook uses the [Deno](https://docs.deno.com/runtime/reference/cli/jupyter/) Typescript engine.
